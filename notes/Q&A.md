### Q1：向量是一个点？求夹角不是应该要两条边吗？

**日期：** 2026-07-05

**问题：**
在 `Word-Embedding.py` 中，每个词（如 `king = [0.9, 0.8]`）看起来只是一个点，但余弦相似度要算两个向量的夹角，夹角不是需要两条边吗？

**解答：**

向量不是"点"，而是**从原点 O(0,0) 指向该点的有向线段（箭头）**。

| 概念 | 实际含义 |
|------|---------|
| 一个 `[0.9, 0.8]` 看起来是一个点 | 它同时也是**从原点指向该点**的箭头 |
| 求夹角的两条边从哪来？ | **原点→king** 是一条边，**原点→queen** 是另一条边 |
| 余弦相似度在算什么？ | 算这两个箭头**方向有多一致**（夹角越小越相似） |

两条边的公共起点就是**原点 O**，所以夹角自然存在。

可视化图片：`notes/images/word-embedding/vector_visualization.png`

- king 与 queen 的夹角 θ ≈ 29.1°，余弦相似度 ≈ 0.874
- `king - man + woman = [0.9, 0.2]` 与 queen 完全重合，夹角 = 0°，相似度 = 1.0

### Q2：大模型术语中，"梯度"是啥意思？

**日期：** 2026-07-05

**问题：**
训练大模型时总说"梯度下降""梯度爆炸""梯度消失"，梯度到底是什么？

**解答：**

梯度就是 **误差的"坡度"**——衡量模型参数往哪个方向调整、调多大，能让损失（loss）降得最快。

| 类比 | 说明 |
|------|------|
| 你在山上下山 | 模型训练就是找"最低点"（损失最小） |
| 脚下斜坡的方向 | **梯度方向**——往哪走坡度最陡 |
| 斜坡的陡峭程度 | **梯度大小**——步子该迈多大 |
| 逆着梯度走一步 | **梯度下降**——往坡底（最优参数）靠近一步 |
| 一步迈太大直接滚下山 | **梯度爆炸**——参数跳飞，loss 变成 NaN |
| 走到平地感觉不到坡 | **梯度消失**——参数几乎不动，模型学不动了 |

**那"损失函数"是什么？**

损失函数（Loss Function）是衡量模型预测结果与正确答案之间差距的"记分员"。

- 预测越准 → loss 越小（得分高）
- 预测越离谱 → loss 越大（得分低）
- 大模型常用 **交叉熵损失（Cross-Entropy Loss）**：预测下一个词时，给正确词概率打高分，其他词打低分，分数越偏 loss 越高

**梯度就是损失函数的"坡度"**——先算出 loss（错多少），再算梯度（往哪调），循环往复直到 loss 降到最低。

**数学上：** 梯度是损失函数对每个参数的偏导数的向量。

- 每个参数（权重 w）都有一个梯度值，正 → 减小该参数；负 → 增大该参数
- 参数更新公式：`w_new = w_old - 学习率 × 梯度`
- LLM 有数十亿参数，反向传播（backpropagation）一次性算出所有参数的梯度，然后统一更新

**一句话：** 梯度告诉模型"你现在错在哪、差了多少、该怎么调"。
