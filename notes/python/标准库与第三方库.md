## Q1：`Fore.YELLOW` 是什么？

**日期：** 2026-07-14

`Fore.YELLOW` 来自第三方库 **`colorama`**，用于在终端中**输出彩色文字**。

### 来源

```python
from colorama import Fore
```

`colorama` 是一个跨平台终端颜色库，支持 Windows / macOS / Linux。

### 基本用法

```python
from colorama import Fore, Back, Style, init

init()  # 初始化（Windows 需要，macOS/Linux 自动生效）

print(Fore.RED + "红色文字")
print(Fore.GREEN + "绿色文字")
print(Fore.YELLOW + "黄色文字")
print(Fore.BLUE + "蓝色文字")
print(Fore.MAGENTA + "紫色文字")
print(Fore.CYAN + "青色文字")

print(Style.RESET_ALL + "恢复默认颜色")  # 重置颜色
```

### 颜色对照表

| 属性 | 颜色 |
|---|---|
| `Fore.BLACK` | 黑色 |
| `Fore.RED` | 红色 |
| `Fore.GREEN` | 绿色 |
| `Fore.YELLOW` | 黄色 |
| `Fore.BLUE` | 蓝色 |
| `Fore.MAGENTA` | 紫色 |
| `Fore.CYAN` | 青色 |
| `Fore.WHITE` | 白色 |
| `Fore.RESET` | 重置为默认 |

### 在 `DigitalBookWriting.py` 中的实际用法

```python
from colorama import Fore

# 用黄色输出任务描述
print(Fore.YELLOW + f"协作任务:\n{task_prompt}\n")

# 用蓝色和绿色区分两个角色的对话
print_text_animated(Fore.BLUE + f"作家:\n\n{user_response.msg.content}\n")
print_text_animated(Fore.GREEN + f"心理学家:\n\n{assistant_response.msg.content}\n")

# 用紫色标识任务完成
print(Fore.MAGENTA + "✅ 电子书创作完成！")

# 用黄色输出统计信息
print(Fore.YELLOW + f"总共进行了 {n} 轮协作对话")
```

不同颜色用来**区分不同角色的输出**，让终端阅读更清晰。

### 背后的原理

终端通过 **ANSI 转义序列**（Escape Codes）来控制颜色：

```
\033[33m  黄色文字  \033[0m
 ↑          ↑         ↑
 转义码    颜色码    重置码
```

`colorama` 帮你封装了这些转义码，直接用 `Fore.YELLOW` 等属性即可，不用记数字。

### 一句话

`Fore.YELLOW` 是 `colorama` 库提供的**黄色文字标记**，加到字符串前面输出到终端就能显示黄色，用来让终端输出更好看、更容易区分不同信息。
